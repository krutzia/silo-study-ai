# Silo – Smart Study Planner

AI-powered smart study planner with adaptive scheduling, analytics, and productivity tools.

## Features

- AI-generated personalized study plans
- Adaptive scheduling that adjusts to your progress
- Social accountability with friends
- Gamification with streaks and badges
- Focus timer with Pomodoro technique
- Rich analytics and progress tracking
- Study chat assistant

## Tech Stack

- React + TypeScript + Vite
- Supabase (Backend & Auth)
- Tailwind CSS + shadcn/ui
- Framer Motion (Animations)

## Getting Started

1. Clone the repository
2. Install dependencies: `bun install`
3. Set up Supabase project and update environment variables
4. Run development server: `bun run dev`

## Environment Variables

Create a `.env.local` file with:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## Build

```bash
bun run build
```
